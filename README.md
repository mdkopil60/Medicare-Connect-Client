# 🏥 MediCare Connect - Client

A modern Hospital Appointment & Healthcare Management System built with Next.js, Tailwind CSS, HeroUI, Better Auth, Stripe, Framer Motion, and Recharts.

# 🔑 Environment Variables

Create a `.env.local`

```env
NEXT_PUBLIC_API_URL=

NEXT_PUBLIC_BETTER_AUTH_URL=

NEXT_PUBLIC_GOOGLE_CLIENT_ID=

NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=
```

# ⚙ Installation

```bash
git clone https://github.com/mdkopil60/Medicare-Connect-Client

npm install

```
